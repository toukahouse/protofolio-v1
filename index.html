Portofolio Proyek

Kumpulan proyek yang telah saya kembangkan untuk mengaplikasikan dan memperdalam keahlian saya di bidang teknologi.

Website Portfolio
HTML CSS Bootstrap

Pengembangan sebuah website portofolio personal dari nol untuk menampilkan profil dan proyek-proyek saya. Dibangun menggunakan HTML, CSS, dan diimplementasikan dengan framework Bootstrap untuk memastikan desain yang responsif dan modern.

Aplikasi Perpustakaan
Java NetBeans

Proyek akademis untuk merancang dan membangun sebuah sistem manajemen perpustakaan desktop. Aplikasi ini menerapkan fungsionalitas dasar database (CRUD - Create, Read, Update, Delete) dan dikembangkan sepenuhnya menggunakan bahasa pemrograman Java dengan IDE NetBeans.

Eksplorasi Machine Learning: Prediksi Saham
Python Belajar ML

Sebuah proyek eksplorasi untuk mempelajari penerapan Machine Learning pada kasus prediksi harga saham. Proyek ini melibatkan analisis data historis dan pembuatan model prediksi sederhana menggunakan Python dengan library seperti Pandas dan Scikit-learn.

Chat Bot AI
HTML CSS JavaScript Python

Sebuah proyek pengembangan chatbot cerdas yang dirancang untuk dapat melakukan percakapan roleplay secara dinamis dan kontekstual. Proyek ini memanfaatkan kekuatan Google's Gemini API untuk menghasilkan respons teks yang kreatif dan sesuai dengan persona karakter yang ditentukan. Fokus utama proyek ini adalah pada integrasi API, rekayasa prompt (prompt engineering) untuk mengarahkan perilaku AI, dan mengelola alur percakapan agar tetap koheren.
